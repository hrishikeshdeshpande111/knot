Knot Commune Website
This is a HTML and CSS file for a website called "Knot Commune", which is a student community platform.

Requirements
To run this website, you need a web browser like Google Chrome or Mozilla Firefox.

How to use
You can visit this link: https://hrishikeshdeshpande111.github.io/knot/
OR
Clone or download the repository to your local machine.
Open the index.html file in a web browser to view the website.

File structure
The general.css file contains general styles for the website.
The header.css file contains styles for the website header.
The page1.css file contains styles for the first section of the website.
The page2.css file contains styles for the second section of the website.
The page3.css file contains styles for the third section of the website.
The page4.css file contains styles for the fourth section of the website.
The page5.css file contains styles for the fifth section of the website.
